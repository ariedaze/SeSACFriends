### 커밋 메시지 
- 커밋 메시지 작성 시 본문에 작업 내용을 꼭 적는다.
- 커밋 메시지 Convention
    - [Feat] : 기능추가 / 새로운 로직
    - [Fix] : 버그수정
    - [Chore] : 간단한 수정
    - [Docs] : 문서 및 리드미 작성
    - [Merge] : 머지, 충돌
    - [Refactor] : 리팩토링
